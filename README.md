# FoodDeliveryApp
 Built a full-stack web application using Java JEE, JSP, Servlets, HTML, CSS, JavaScript, and MySQL.
– Implemented user authentication, menu display, cart system, and order placement features.
– Developed admin modules for restaurant, menu, and order management.
– Designed a responsive UI with smooth animations to enhance user experience.
– Used MVC architecture for clean and maintainable application structure.
– Integrated MySQL for efficient CRUD operations and relational data handling.
– Deployed and tested the application on Apache Tomcat.
– Technology Used: Java JEE, JSP, Servlets, MVC, MySQL, HTML, CSS, JavaScript, JDBC, Apache Tomcat
